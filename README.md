## Запуск
Перед запуском проекта установите необходимые зависимости 
```bash
  pip install -r requirements.txt
```

## Используемые технологии
* Flask
* SQLAlchemy
* Werkzeug
* WTForms
* ORM база данных со связями
* bootstrap
